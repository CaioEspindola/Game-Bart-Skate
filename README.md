# Game Skate with Bart Simpson.

Programming logic with javaScript.

Features:

- This application used the P5JS editor: https://editor.p5js.org/
- Game theme: Skate and Bart Simpson.
- Commands to play: "w,s,a,d" and "Enter".

Enjoy!!
